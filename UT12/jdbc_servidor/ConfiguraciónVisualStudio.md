# CONFIGURACIÓN VISUAL STUDIO CODE COMO  ENTORNO WEB INTEGRADO
A continuacion se  describen los pasos esenciales para configurar un entorno de desarrollo en Visual Studio Code.

Explica cómo preparar el entorno para trabajar con SQLite como base de datos embebida, detallando la instalación de plugins necesarios y la configuración de dependencias en Maven. 

Además, encontrarás  la estructura del proyecto ideal para alojar la base de datos y  y la  orientación sobre cómo determinar la ruta de la BBDD cuando se utiliza un servidor virtual como Red Hat Server Connectors con Tomcat.
